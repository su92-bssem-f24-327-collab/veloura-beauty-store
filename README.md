# 💄 Veloura Beauty Store

A modern and responsive mini e-commerce beauty store built using HTML, CSS, and JavaScript.

## 📌 Project Overview

Veloura Beauty Store is a frontend e-commerce project developed as part of an internship assignment. The project simulates a real-world online beauty store where users can browse products, view product details, and manage a shopping cart.

## 🚀 Features

* Dynamic Product Listing
* Product Detail Page
* Category Filtering
* Price Sorting
* Product Search
* Shopping Cart Functionality
* Quantity Management
* LocalStorage Integration
* Real-Time Cart Updates
* Responsive Design
* Modern UI & Smooth Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage
